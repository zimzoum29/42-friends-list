# 42 Intra Friends

This repo contains a web extension designed for Brave, allowing you to have shortcuts to your friends' profiles on your intra 42 profile.

## What is the extension used for ?

This extension adds a panel to your intranet home page so you can add your friends' names and access them quickly.


## Download

To install this extension, you have just to download the directory 42-friends-list/ and add it as an extension in your browser.

## Warning

The extension was developed for use on Brave; proper functioning of this extension on other browsers is not guaranteed !

The extension was coded with AI, so if any parts of the code seem inconsistent to you, please contact me to discuss it. I would be delighted to learn new things or new ways to optimize the extension.
